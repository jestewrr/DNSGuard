DNSGuard Browser Extension

Protect your browsing in real-time. The extension automatically scans every website you visit and blocks malicious domains before they can harm you.

Download Extension (.zip)

Compatible with Google Chrome & Chromium-based browsers

Why isn't this on the Chrome Web Store?

Publishing to the Chrome Web Store requires a $5 developer fee and a manual review process by Google that can take days or weeks. For academic and development purposes, sideloading the extension via Developer Mode is the standard and recommended approach. This method provides the exact same functionality and security as a Web Store extension.

Installation Guide

1

Download the Extension

Click the "Download Extension" button above. A DNSGuard_Extension.zip file will be saved to your Downloads folder.

2

Extract the ZIP File

Right-click the downloaded .zip file and select "Extract All...". Remember where you extracted it.

3

Open Chrome Extensions Page

Open Google Chrome and type chrome://extensions in the address bar, then press Enter.

4

Enable Developer Mode

In the top-right corner of the Extensions page, toggle the "Developer mode" switch to ON.

5

Load the Extension

Click the "Load unpacked" button that appears. Navigate to the folder where you extracted the ZIP file, select it, and click "Select Folder".

✓

Done! Pin It to Your Toolbar

The DNSGuard icon will appear in your Extensions menu. Click the puzzle icon in Chrome's toolbar and click the pin icon next to DNSGuard to keep it visible at all times.

What the Extension Does

Real-Time Scanning

Every website is analyzed before it loads using DNS checks, pattern analysis, and reputation scoring.

Automatic Blocking

Malicious and suspicious sites are immediately blocked with a full-screen warning before any harm can occur.

Activity Logging

All browsing activity is logged to your personal dashboard for review and threat accountability tracking.
